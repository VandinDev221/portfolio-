Entendendo o JavaScript Assíncrono

Data: 5 de março de 2025

O JavaScript assíncrono pode parecer complicado no começo, mas é uma das partes mais poderosas da linguagem...

O que é JavaScript Assíncrono?

JavaScript é uma linguagem de programação síncrona...

Callbacks

            
<script>
function fetchData(callback) {
    setTimeout(() => {
        const data = "Dados carregados!";
        callback(data);
    }, 2000);
}

fetchData(function(data) {
    console.log(data);  // Exibe: "Dados carregados!"
});
</script>

Promises

            
function fetchData() {
    return new Promise((resolve, reject) => {
        setTimeout(() => {
            const data = "Dados carregados!";
            resolve(data);
        }, 2000);
    });
}

fetchData().then((data) => {
    console.log(data);  // Exibe: "Dados carregados!"
}).catch((error) => {
    console.error(error);
});

Async/Await

            
<script>
async function fetchData() {
    const data = await new Promise((resolve, reject) => {
        setTimeout(() => {
            resolve("Dados carregados!");
        }, 2000);
    });
    console.log(data);  // Exibe: "Dados carregados!"
}

fetchData();
</script>

Conclusão

O JavaScript assíncrono é essencial para criar aplicações web rápidas...
